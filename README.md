## How to run

### Prerequisites

- install `requirements.txt` via
  
  ```bash
  pip install -r requirements.txt
  ```

- run sample scripts in `src/argv` using

    ```bash
    bash cmd/run_sample_parser.sh
    ```

- run sample scripts in `src/argparse` using

    ```bash
    bash cmd/run_sample_argparse.sh
    ```
